# Evaluation

Evaluate the quality and structure of outputs generated by the AI Structured Output Engine.

---

## Setup

Ensure the main application is installed and environment variables are configured.

---

## How to Run Evaluator

```
python -m evaluation.evaluator
```

---

## How to Run Optimizer

```
python -m evaluation.optimizer
```

---

## Input Required

Test cases are defined in:

```
evaluation/test_cases.json
```

Each test case includes:

* input text
* schema definition

---

## Sample Output

```
Input: Login fails randomly
Output: {
  "problem": "...",
  "root_causes": [...],
  "impact": "...",
  "priority": "High"
}
```

---

## Evaluation Summary

The evaluator checks:

* Structure validity (all schema keys present)
* Basic output consistency

---

## Notes

* This evaluation is structural and does not validate semantic correctness
* Extend evaluator to include LLM-based judging for deeper validation
* Optimizer can be extended to iteratively improve prompts based on failures

---

Built to support continuous improvement of structured output generation
